# DNS — интернет-магазин электроники и бытовой техники

## Описание

Проект представляет собой одностраничное приложение (SPA) интернет-магазина электроники и бытовой техники под брендом «Днс». Реализован каталог товаров с фильтрацией, корзина с сохранением в localStorage, форма оформления заказа с валидацией, lazy loading карточек и адаптивный дизайн.

## Стек технологий

- HTML5 (семантическая верстка)
- CSS3 (Flexbox, Grid, анимации, media queries)
- JavaScript (ES6+, модульная архитектура)
- Font Awesome (иконки)
- GitHub Pages (хостинг)

## Как запустить

1. Клонируйте репозиторий:
   bash
   git clone https://github.com/your-username/dns-agvd.git
   cd dns-agvd
   

2. Откройте `index.html` в браузере (можно через Live Server).

3. Для публикации на GitHub Pages:
   - В настройках репозитория → Pages → Source: Deploy from branch → main → / (root) → Save.
   - Сайт будет доступен по адресу: `https://your-username.github.io/dns-agvd/`

## Основные функции

- Каталог товаров с фильтрацией по цене, бренду, категории
- Поиск по названию
- Корзина с подсчетом суммы, изменением количества и удалением
- Форма оформления заказа с валидацией
- Lazy loading карточек товаров
- Анимации: плавное появление, hover-эффекты, микро-взаимодействия
- Адаптивный дизайн (320px–1920px)
- Доступность: семантическая верстка, alt-тексты, контраст
